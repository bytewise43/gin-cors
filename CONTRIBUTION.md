# Contribution Guidelines

You want to contribute to this project? Great! Please read the following guidelines before you start.

## How to contribute
- Fork the repository and create a new branch for your feature or bug fix.
- Make your changes and commit them with clear and descriptive messages.
- Push your changes to your forked repository and create a pull request to the main repository.
- Please make sure to follow the coding style and conventions used in the project. -> Use the provided commands and the pre-commit hooks to ensure your code is properly formatted and linted.
- If your pull request is related to an issue, please reference the issue number in your pull request description.
